> 基データ: 競合15社・約30プラン調査 + Web市場リサーチ

## エグゼクティブサマリー

### 結論
**「士業特化 × ハンズオン実装 × 3ヶ月卒業型」のAI導入プログラムで参入する。**

| 項目 | 内容 |
|------|------|
| ターゲット | 税理士事務所（第一弾）→ 社労士・弁護士に横展開 |
| サービス名案 | 「3ヶ月AI導入プログラム for 士業」 |
| 月額 | 15万円（税抜）× 3ヶ月 = 総額45万円 |
| コアバリュー | 書類作成業務30%削減を3ヶ月で実現 |
| 差別化 | 士業特化（競合ゼロ）+ 実装代行 + 成果コミット + 卒業型 |
| 初年度目標 | 15社契約、年商約1,000万円 |

## 1. 市場機会

### なぜ「今」か

```
① 市場の立ち上がり期
   - 日本の生成AI市場: 2025年 約8,850億円 → 2026年 約1.4兆円（+58%成長）
   - 中小企業のAI導入率はまだ5-10%。伸びしろが巨大

② 中小企業の「AI焦燥感」がピーク
   - 41.9%が「用途がわからない」
   - 「何もしない」への不安が高まっている
   - 同業他社の導入事例がニュースになり始めた

③ 競合が「全業種・相談型」に固まっている
   - 15社中14社が全業種対応
   - 差別化困難 → 価格競争（「激安」「爆安」の乱立）
   - 業種特化の空白地帯が広大
```

## 2. ターゲット選定

### 推奨ターゲット: 士業事務所（税理士を最優先）

| 選定理由 | 詳細 |
|---------|------|
| 競合が完全空白 | 15社中0社が士業特化 |
| 「顧問」モデルに馴染みがある | 税理士自身が企業の「顧問」をしている業界 |
| AI活用との相性が高い | 書類作成・情報収集・顧客対応 = AIの得意領域 |
| 支払能力が安定 | 顧問報酬モデルで収入安定。月額15万円は許容範囲 |
| 口コミ拡散力 | 士業同士のネットワークが濃密 |
| 危機感が強い | 「AIに仕事を奪われる」議論が活発 → 導入動機が強い |
| 市場規模 | 税理士事務所約2.6万 + 社労士約2.7万 + 弁護士約1.8万 = 7万超 |

### 横展開ロードマップ
```
Phase 1: 税理士事務所（6ヶ月）
Phase 2: 社労士・行政書士（+6ヶ月）
Phase 3: 弁護士事務所（+6ヶ月）
Phase 4: 建設業・製造業（2年目〜）
```

## 3. 顧客が本当に求めているもの

### ジョブ理論の核心

```
顧客が雇いたいのは「AI顧問」ではない。
「AIで確実に成果を出してくれる、業界を理解したガイド」である。

具体的には:
✅ 自分の業界・業務を理解している
✅ 何をすべきか明確に教えてくれる
✅ 実際にやってくれる（設定・カスタマイズ代行）
✅ 成果が数字で見える
✅ 最終的には自分たちでできるようになる

既存15社が提供しているのは上2つだけ。
下3つが圧倒的に不足 → ここが勝機。
```

### 購買トリガー TOP3
1. 同業事務所がAI導入して「繁忙期なのに定時で帰れるようになった」と聞いた
2. 電帳法・インボイス等の法改正対応に追われ「もう手作業では限界」と感じた
3. 顧問先から「AIで何かできないか」と相談され、答えられなかった

## 4. 競合に勝つ価値提案

### ポジショニング: 唯一の「業種特化 × 実装支援型」

```
                    業種特化
                      ↑
                      |
          ★ 本サービス |
                      |
  ────────────────────┼──────────→ 実装支援の深さ
                      |
   ● 激安AI顧問       |
   ● 爆安AI顧問       |  ● Aidia
   ● アドカル         |
          ● ギブリー  |
          ● エッジ    |
                      ↓
                    全業種対応

→ 右上象限はほぼ空白。先行者利益を確保できる。
```

### 4つの差別化ポイント

| # | 差別化 | 既存競合 | 本サービス |
|---|--------|---------|-----------|
| 1 | **士業特化** | 全業種対応（浅い知識） | 税理士業務を深く理解（テンプレート付き） |
| 2 | **ハンズオン実装** | MTGで相談して終わり | AIツールの設定・カスタマイズまで代行 |
| 3 | **成果コミット** | 「支援します」止まり | 「書類作成30%削減」を約束 |
| 4 | **卒業型プログラム** | 永続サブスク（依存構造） | 3ヶ月で自走を目指す（信頼構築） |

## 5. サービス設計

### メインプラン: 3ヶ月AI導入プログラム

| 項目 | 内容 |
|------|------|
| 月額 | 15万円（税抜） |
| 期間 | 3ヶ月（卒業型） |
| 総額 | 45万円（税抜） |
| MTG | 月2回（各60分・オンライン） |
| チャット | 無制限（Slack/ChatWork） |
| 実装支援 | AIツール設定・カスタマイズ代行（月4時間） |
| 成果物 | AI活用マニュアル、業務フロー改善書、効果測定レポート |
| KPI | 対象業務の工数30%削減 |
| 補助金 | IT導入補助金で実質50%OFF（22.5万円）可能 |

### 3ヶ月のプログラム構成

```
Month 1: 診断・設計
  → 現状業務棚卸し → AI適用業務TOP5特定 → 第1業務に導入

Month 2: 実装・展開
  → 第2・第3業務に展開 → 全スタッフ研修

Month 3: 定着・卒業
  → 自走テスト → 効果測定 → 卒業認定
```

### 卒業後オプション
- 月額5万円のライトサポート（月1回MTG + チャット）
- 推定継続率: 70%

## 8. 成功のためのクリティカルサクセスファクター

### 絶対に外してはいけない5つ

| # | CSF | 理由 |
|---|-----|------|
| 1 | **MVP顧客で実証データを作る** | 「30%削減」の実績がなければ全てが空論 |
| 2 | **士業業務の深い理解** | 浅い知識では「全業種対応」の既存競合と同じになる |
| 3 | **実装まで含む支援** | 「相談だけ」では既存競合との差別化が効かない |
| 4 | **卒業後の継続率70%以上** | LTV/CACの健全性がここにかかっている |
| 5 | **紹介による顧客獲得** | 士業ネットワークを活かせればCAC大幅削減 |

### 最大のリスクと対策

| リスク | 対策 |
|--------|------|
| 3ヶ月で成果が出ない | 対象業務を「議事録作成」1つに絞り、確実に70%削減を見せる |
| 士業のAI抵抗感 | 「AIに代替される」ではなく「AIで付加価値を上げる」メッセージ |
| 競合が士業特化で参入 | 早期に事例・ノウハウ・コミュニティを蓄積し、スイッチングコストを構築 |
