# STEP 5: 収益モデル・ユニットエコノミクス設計

## 1. 価格戦略の決定

### 3つのアプローチによる検証

#### ① コストベース（原価積み上げ）
```
コンサルタントの時間単価: 1.5万円/時
月間稼働時間/顧客: 約8時間（MTG2h + 実装4h + 準備2h）
原価: 1.5万 × 8h = 12万円
目標粗利率: 40%
→ 月額: 12万 ÷ 0.6 ≒ 20万円
```

#### ② 競合ベース（市場相場準拠）
```
ボリュームゾーン: 月額10万円
差別化要因（士業特化 + ハンズオン実装）のプレミアム: +50%
→ 月額: 15万円
```

#### ③ バリューベース（顧客価値から逆算）★採用
```
税理士事務所の書類作成業務: 月間約80時間（スタッフ2-3人分）
AI導入による削減: 30%（PwC実証データ） = 24時間/月
時間単価: 2,500円（スタッフ人件費）
月間削減価値: 24h × 2,500円 = 6万円
+ 残業代削減・ミス防止等の間接効果: 約4万円
合計顧客価値: 約10万円/月

価格 = 顧客価値の1.5倍（初期投資回収後のROI含む）
→ 月額: 15万円（3ヶ月プログラム = 総額45万円）
```

### 価格決定: 月額15万円（税抜）/ 3ヶ月プログラム

**根拠:**
- ボリュームゾーン（10万円）より上だが、ギブリー（33万円）の半額以下
- 「安さ」ではなく「成果」で選ばれるポジション
- 3ヶ月で45万円 = 税理士の顧問先1社の年間顧問料に相当 → 投資判断しやすい
- 補助金（IT導入補助金）活用で実質22.5万円にできる

## 3. ユニットエコノミクス

### 1顧客あたりの収益構造

```
【初期3ヶ月（プログラム期間）】
売上:        15万円 × 3ヶ月 = 45万円
変動費:      コンサルタント稼働 8h/月 × 3ヶ月 × 1.5万円/h = 36万円
粗利:        45万 - 36万 = 9万円
粗利率:      20%

【卒業後サポート（継続した場合）】
売上:        5万円/月
変動費:      稼働2h/月 × 1.5万円/h = 3万円
粗利:        2万円/月
粗利率:      40%
平均継続:    6ヶ月（推測）
追加粗利:    2万 × 6ヶ月 = 12万円

【LTV（顧客生涯価値）】
= 初期粗利9万 + 追加粗利12万 = 21万円
※卒業後未継続の場合は9万円
```

### 顧客獲得コスト（CAC）推計

```
【チャネル別CAC（推測）】
- 紹介・口コミ:     1万円（最も効率的）
- セミナー集客:      5万円（1回30万円で6社獲得想定）
- Web広告:          10万円（CPA）
- コンテンツSEO:     3万円（長期的にCACが下がる）

加重平均CAC:  5万円（推測）
```

### LTV/CAC比率

```
LTV / CAC = 21万円 / 5万円 = 4.2倍 ✅（3倍以上で健全）

※卒業後未継続の場合: 9万円 / 5万円 = 1.8倍 ⚠️
→ 卒業後サポートへの移行率が重要KPI
```

## 4. 損益分岐分析

### 月間固定費（推測）

| 項目 | 月額 |
|------|------|
| コンサルタント人件費（自分） | 50万円 |
| ツール費用（各種SaaS） | 5万円 |
| マーケティング費 | 15万円 |
| その他経費（通信、税理士等） | 5万円 |
| **合計** | **75万円** |

### 損益分岐点

```
1顧客あたり月間粗利:
- プログラム期間中: (15万 - 12万) = 3万円/月
- 卒業後サポート: (5万 - 3万) = 2万円/月

必要顧客数（プログラム顧客のみ）:
75万円 ÷ 3万円 = 25社（同時並行）

→ 1人で月8h/社 × 25社 = 200h/月（不可能）

現実的なキャパシティ:
1人で同時対応可能: 8〜10社
→ 月間売上: 15万 × 10社 = 150万円
→ 月間粗利: 3万 × 10社 + 2万 × 10社(卒業後) = 50万円
→ 営業利益: 50万 - 75万 = ▲25万円（赤字）

黒字化の条件:
- コンサル単価を上げる（月額20万円→粗利8万円）
- 卒業後継続率を高める（20社×2万円=40万円を確保）
- ツール・テンプレートの標準化で変動費を下げる
- 外注パートナーの活用
```

### 現実的な成長シナリオ

| Phase | 期間 | 顧客数 | 月間売上 | 体制 |
|-------|------|--------|---------|------|
| 0（検証） | 1-3ヶ月 | 3社（無料/格安） | 0-15万円 | 1人 |
| 1（立上げ） | 4-9ヶ月 | 5-8社 | 75-120万円 | 1人 |
| 2（安定） | 10-18ヶ月 | 10-15社 | 150-225万円 | 1人+外注1 |
| 3（拡大） | 19-24ヶ月 | 20-30社 | 300-450万円 | 2人+外注2 |

## 5. 収益改善のレバー

### 短期（Phase 0-1）
1. **テンプレート化**: 士業向けAI導入手順書を標準化し、変動費を月6h/社に削減
2. **補助金活用の訴求**: 「IT導入補助金で実質50%OFF」で価格障壁を下げる
3. **紹介プログラム**: 既存顧客からの紹介で1社紹介ごとに月額1万円割引

### 中期（Phase 2-3）
4. **グループプログラム**: 5社同時開催で1社あたり月額10万円、変動費を大幅削減
5. **サブスクリプション型コンテンツ**: 月額3万円のオンラインコース（スケーラブル）
6. **士業向けAIツールの自社開発**: 月額5,000円〜のSaaSで継続収益を創出

### 長期
7. **認定パートナー制度**: 他のコンサルタントにメソッドをライセンス提供
8. **業種横展開**: 士業で確立したモデルを建設業・製造業に適用
