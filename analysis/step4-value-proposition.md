# STEP 4: 競合優位な価値提案の設計

## 1. 業界の「暗黙の前提」を崩す

### 既存15社の共通パターン（崩すべきポイント）

| 業界の常識 | なぜ問題か | 逆張りの方向 |
|-----------|-----------|-------------|
| MTG回数で価格差をつける | 顧客はMTG回数ではなく「成果」が欲しい | 成果指標ベースの価格設計 |
| 全業種対応 | 業界知識が浅く「一般論」しか言えない | 1業種に深く特化 |
| 「相談」がメインの支援 | 相談後のアクションは顧客任せ | ハンズオン実装まで含む |
| 月額定額サブスク | 永久に払い続ける構造 | 卒業ゴールのあるプログラム |
| 人が全部対応 | スケーラビリティがない | AIツール+人のハイブリッド |
| 成果の定義が曖昧 | 「支援します」では顧客は納得しない | 具体的KPIをコミットする |

## 2. バリュープロポジションキャンバス

### ターゲット: 士業事務所（税理士・社労士を最優先）

#### 顧客の「ペイン」
1. AIで何ができるか具体的にわからない（41.9%が「用途不明」）
2. 忙しくて学ぶ時間がない
3. 導入しても社員が使わなくなる
4. セキュリティが心配（顧客情報を扱う）
5. 投資した分の元が取れるか不明
6. 「AIに仕事を奪われる」という漠然とした不安

#### 顧客の「ゲイン」
1. 書類作成時間が30-40%削減される（PwC実証済み）
2. 法改正情報の収集が自動化される
3. 顧客対応の質と速度が向上する
4. スタッフの残業が減る
5. 新たな付加価値サービスを顧客に提供できる
6. 「AI活用事務所」として差別化・採用力向上

#### 自社の「ペインリリーバー」
1. 士業専用のAI活用テンプレート・手順書を提供
2. 初回診断で「あなたの事務所でAIが使える業務TOP5」を即提示
3. 週次の非同期レポートでMTG負荷を最小化
4. 士業向けセキュリティガイドラインを標準提供
5. 導入効果ダッシュボードで毎月のROIを可視化
6. 3ヶ月で「自走できる状態」にする卒業プログラム

#### 自社の「ゲインクリエイター」
1. 実際にAIツールの初期設定・カスタマイズまで代行
2. 士業の成功事例ライブラリ（同業の実績で安心感）
3. 導入企業コミュニティ（士業同士の情報交換の場）
4. 法改正AIアラート（士業特有の付加価値）
5. 顧問先への「AI活用提案テンプレート」提供

## 3. 価値提案ステートメント

### メイン価値提案

```
【ターゲット】 税理士・社労士・弁護士事務所が
【課題】 「AIを導入したいが何から始めればいいかわからない」時に
【サービス】 本サービスは
【独自の方法】 士業専用のAI導入テンプレート + ハンズオン実装 + 効果可視化ダッシュボードで
【成果】 3ヶ月で書類作成業務の30%削減を実現し、自走できる状態にする
```

### サブ価値提案（3つの柱）

| 柱 | 内容 | 競合との差別化 |
|----|------|--------------|
| **1. 士業特化** | 税務申告・社会保険手続き・契約書作成など士業固有の業務にAIを適用 | 15社中0社が士業特化 |
| **2. ハンズオン実装** | MTGで「相談」するだけでなく、AIツールの設定・カスタマイズまで代行 | ほとんどが「相談止まり」 |
| **3. 卒業プログラム** | 3ヶ月で社内が自走できる状態を目指す。永久依存させない | 既存は永続サブスク |

## 5. サービス名・ブランディング案

### 命名方針
- 「激安」「爆安」の価格訴求路線を避ける
- 「顧問」ではなく「パートナー」「導入」「実装」を軸に
- 士業向けであることが一目でわかる

### 候補案（推測・提案）

| 案 | コンセプト | 印象 |
|-----|-----------|------|
| **AI実装パートナー for 士業** | 「実装」という行動を約束 | プロフェッショナル |
| **士業AI Lab** | 士業のAI実験室・研究所 | 先進的・知的 |
| **AI顧問 Pro（士業専門）** | 既存カテゴリの上位互換 | わかりやすい |
| **3ヶ月AI導入プログラム** | 期間と成果を明示 | 具体的・安心 |

→ 「成果にコミットする」「期間限定で自走を目指す」メッセージが最も差別化に効く（推測）
