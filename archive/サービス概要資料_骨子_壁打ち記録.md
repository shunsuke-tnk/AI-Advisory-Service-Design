## 資料の目的と想定読者

### 想定読者

| 読者 | 状況 | 関心事 |
|------|------|--------|
| **経営者（社長・専務）** | セミナー後、または紹介で資料を受け取る | 「うちの会社で本当に使えるのか」「投資対効果」 |
| **現場責任者・部長クラス** | 社長から検討を任される | 「具体的に何をしてくれるのか」「現場の負担は増えないか」 |
| **紹介者** | 知人の経営者に渡す | 「簡潔に価値が伝わるか」「渡しやすいか」 |

### 資料を読む場面

1. セミナー参加後、社内検討のため持ち帰る
2. 紹介者から「これ見てみて」と渡される
3. 商談前の事前送付資料として
4. 社内稟議の添付資料として

### 資料のゴール

**「無料相談・無料診断を申し込む」**

資料だけで契約を決めさせるのではなく、「一度話を聞いてみよう」と思わせることがゴール。

> 医療、美容、士業など、これまで1000人以上の経営者をサポートしてきました。
>
> その中でも、私が**真っ先に変えたい**と思っているのが建設業界です。
>

## 価格の見せ方

### 「AI人材を雇う」という比較軸の活かし方

**NG**：他のコンサルティングサービスとの価格比較
**OK**：「AI人材を採用した場合」との比較

**理由**：
- 競合は「汎用AIコンサル」より「何もしない（独学）」「人を雇う」
- 「月10万円」を「高い」と感じさせないためには、比較対象を変える必要がある

### 価格表示の工夫

**視覚的比較（資料内に入れる図）**：

```
┌─────────────────────────────────────┐
│ AI人材を正社員採用                    │
│ 年間 600〜1,000万円                  │
│ + 採用コスト・教育コスト              │
│ + 退職リスク                         │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│ 本サービス（竹プラン）                 │
│ 年間 120万円（月10万円）              │
│ 採用不要・すぐ開始・いつでも相談可能   │
└─────────────────────────────────────┘
```

### 「月10万円の価値」の伝え方

**時間換算**：
- Zoom月2時間 + チャット無制限
- 実質「週1回相談できるAI顧問がいる状態」
- 時給換算ではなく「安心感」「何かあったときに頼れる」という価値

**成果換算**：
- 見積もり作成が2時間→10分になれば、月20件で約40時間削減
- 時給3,000円換算で月12万円の削減効果

## 壁打ちで決定した事項

### 0. 競合の定義（最重要）
- **競合は「他のAIコンサルサービス」ではなく、「AI人材を雇用する」という選択肢**
- 読み手の頭の中にある比較対象は「正社員を雇う」「パートを雇う」という雇用
- だから「月10万円」という価格が、雇用コストと自然に比較される表現が必要
- 「顧問」という言葉だけでは、雇用の代替という比較軸が薄れてしまう
- キャッチコピーを見た瞬間に「雇用するより安い」と直感的に伝わることが重要

### 1. 資料のボリューム・形式
- **基本資料**：4〜6ページ（紹介・セミナー配布・PDF送付用）
- **詳細版**：必要に応じて別途作成

### 2. ストーリーの出し方
- **詳しく書く**：原体験ストーリーで共感を取る
- 「苦労→乗り越え→バリュー」の軸
- 資料全体の分量を見ながら具体性を調整

### 3. 価格の見せ方
- **顧問プラン / 参謀プラン**の2プラン構成
- プラン名案（A〜D）は保存済み

### 4. 差別化の強調度合い
- **余地を残す**：他業種実績も示しつつ、建設業への思いは強く出す
- トーン：中〜強（「真っ先に変えたい」レベル）
- 「恩返し」の文脈は入れない

### 5. CTAの設計
- **無料相談（30分Zoom）一本**
- 名称：無料AI活用診断
- 売り込みなし、合う・合わないを確認するプチコンサル形式

## サービス名・キャッチフレーズの検討

### セッション2（2026年1月30日）：サービス名の壁打ち

#### 名称設計の条件
1. **「顧問」を前面に** → 競合は雇用だから
2. **業界はサブ要素** → 「〇〇業界向け」は後付けで切り替え可能に
3. **横展開可能** → 建設業、介護業界、人材業界など業界を差し替えても成立する構造

#### 検討した方向性

**採用するニュアンス**：
- 「顧問」「参謀」 → 経営に近い存在、任せられる感
- 「パートナー」「伴走」 → 一緒に進む関係性
- **「雇っている感覚」が出ること**が重要

**NGなニュアンス**：
- 安っぽさ（レンタル、シェア）
- 単なる専門家・外注感
- 部署名っぽさ
- 横文字・造語系（響きが微妙）

#### 最終候補（4案）

| 候補 | 補足 |
|------|------|
| **AI顧問** | シンプル、説明不要、覚えやすい。やや一般的だが差別化はサブコピーで補う |
| **AI参謀** | 戦略的、知的な印象、経営者に刺さる。やや硬いが格がある |
| **専属AI顧問** | 「あなた専用」感が強い。少し長いが安心感がある |
| **あなたのAI右腕** | イメージしやすい、親しみやすい。「社長の右腕」という馴染みある表現 |

#### 方針決定
- **メインサービス名は「AI顧問」をベースに**
- キャッチフレーズ（サブコピー）で魅力・差別化を補足する方針
- 業界名は「for 建設業」「建設業向け」など後付けで対応

#### 次のステップ
